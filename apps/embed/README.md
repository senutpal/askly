# Askly Embed Script

The lightweight loader script for the Askly widget.

## Overview

This package generates the `widget.js` file that clients include on their websites. It:
- Creates the iframe
- Handles communication between the host page and the widget iframe
- Manages widget visibility (open/close)

## Tech Stack

- **Build Tool**: Vite
- **Language**: TypeScript

## Scripts

- `pnpm dev`: Start development server
- `pnpm build`: Build for production
