# Askly Backend

The backend infrastructure for Askly, powered by Convex.

## Overview

Handles all server-side logic, database operations, and AI integrations.

## Features

- **Database**: Real-time data sync
- **Vector Search**: RAG implementation for knowledge base
- **Actions**: Integrations with OpenAI, Gemini, Vapi
- **Auth**: Clerk integration

## Scripts

- `pnpm dev`: Start Convex dev server
- `pnpm run deploy`: Deploy to production
